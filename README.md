# RA3-y-RA4.-AF-Uso-de-Arrays-y-manipulaci-n-del-DOM
Este proyecto es una guía interactiva de los 39 métodos de Array de JavaScript.

## Cómo usarlo
Simplemente abre el archivo `index.html` en cualquier navegador web moderno.

## Estructura
El proyecto aplica el Principio de Responsabilidad Única (SRP) separando la estructura de datos de la lógica de renderizado del DOM.
